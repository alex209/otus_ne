# Лабораторная №7

## IS-IS.

### Цели задания

- Настроить IS-IS офисе Триада

### Топология сети

![](./img/lab_07.png)

### Задачи

- Настроите IS-IS в ISP Триада.
- R23 и R25 находятся в зоне 2222.
- R24 находится в зоне 24.
- R26 находится в зоне 26.

Настройка осуществляется одновременно для IPv4 и IPv6.
